Common CSS Properties Reference

CSS Properties

Property Description Syntax
font-family Sets the font of specified text. font-family: Helvetica;
width Sets the width of an element. width: 150px;
margin Sets the margin area on all four sides of an elemnt. margin: 10px auto;
border Sets an element's border values (width, style, color). border: dashed red;
padding Sets the padding area on all four sides of an element. padding: 10px 50px 30px 0;
float Places an element on the left or right side of its container. float: left;
display Sets whether an element is treated as a block or inline element. display: block;
color Sets the color of an element's text and text decorations. color: blue;
background sets all background style properties at once. background: border-box red;
